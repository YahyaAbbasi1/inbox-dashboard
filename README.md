# Frontend Assessment - Inbox Dashboard

A modern inbox dashboard built with React + TypeScript, featuring a honeycomb loading animation, skeleton loaders, chat interface, and API integration.

## Features

- ✅ **Loading States** - Honeycomb animated loader followed by skeleton loader
- ✅ **Sidebar Navigation** - Teams, Users, and Channels sections
- ✅ **Chat List** - Search and filter conversations
- ✅ **Chat Window** - Message bubbles with typing support
- ✅ **Details Panel** - Contact info, labels, notes, assignment
- ✅ **API Integration** - Real data from DummyJSON API
- ✅ **Responsive Design** - Adapts to different screen sizes
- ✅ **TypeScript** - Full type safety

## APIs Used

- **DummyJSON Users** - `https://dummyjson.com/users` - For chat contact list
- **DummyJSON Posts** - `https://dummyjson.com/posts` - For message history

## Setup Instructions

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd inbox-dashboard

# Install dependencies
npm install

# Start development server
npm start
